# Security Policy

## Supported Versions
This project is continuously updated. Security fixes are applied as needed.

## Reporting a Vulnerability
If you discover a security issue or vulnerability, please email:

**security@studioloft.media**

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Any potential impact or severity

You will receive acknowledgement within **48 hours**.

Please **do not open a public GitHub issue** for security concerns.
